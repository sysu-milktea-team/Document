### 2. 游戏简介

![game_poster](./image/design/game_poster.png)

这是一款结合AR的模拟经营类游戏。 

玩家扮演一个奶茶店的服务员，向光临奶茶店的顾客提供奶茶。

玩家需要根据顾客的需求，在顾客的耐心时间内，调制出不同口味的奶茶，满足不同顾客的不同需求。
